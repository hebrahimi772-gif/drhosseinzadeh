# Ethics and Governance

The dataset included in this repository has been prepared for the sole purpose of reproducing the results of the associated manuscript. The publicly shared files contain no direct identifiers; organizational identifiers are anonymized and individuals are pseudonymous.

Users are expected to exercise responsible conduct when handling the data. The dataset is provided under terms that allow academic evaluation and reproducibility; any further use should respect participant confidentiality and the original study’s ethical approvals.

There are no legal claims made by the authors; the repository serves to support transparent and verifiable research.

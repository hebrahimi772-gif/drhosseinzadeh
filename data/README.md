# Data Directory

This folder contains the raw dataset and associated metadata used in the manuscript.

- `raw/ocmcap_raw_dataset_v1.csv` – primary anonymized survey data.
- `raw/schema_ocmcap_v1.json` – JSON schema describing variable definitions and dataset structure.
- `metadata/dataset_manifest_ocmcap_v1.csv` – dataset provenance and usage notes.
- `metadata/codebook_ocmcap_v1.csv` – detailed variable codebook.

These files are authoritative and must remain unchanged for reproducibility.
